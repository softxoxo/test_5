## Установка и запуск

1. Клонируйте репозиторий:
```
git clone https://github.com/softxoxo/test_5.git
```
2. Перейдите в директорию проекта:
```
cd test_5
```
4. Установите зависимости:
```
npm install
```
4. Запустите калькулятор:
```
npm run start
```

## Тестирование

Для запуска тестов выполните команду:
```
npm test
```
